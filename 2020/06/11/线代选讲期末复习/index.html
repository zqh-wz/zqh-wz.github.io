欧几里得空间与酉空间

欧几里得空间

设$V$是实数域上的线性空间，若对于$V$中任意两个元素$\alpha, \beta$，都定义了内积运算$(\alpha, \beta)$，且满足：

  1. $(\alpha, \beta) = (\beta, \alpha)$
  2. $(k \alpha, \beta) = k(\alpha, \beta)$
  3. $(\alpha + \beta, \gamma) = (\alpha, \gamma) + (\beta, \gamma)$
  4. $(\alpha, \alpha) \ge 0$，且$(\alpha, \alpha) = 0 \Leftrightarrow \alpha = \theta$

则称$V$是欧几里得空间。

欧氏空间的例子：$n$维向量空间$\mathbb R^n$

可以定义长度$||\alpha|| = \sqrt{(\alpha, \alpha)}$与夹角$<\alpha, \beta> = arccos \frac{(\alpha, \beta)}{||\alpha|| \cdot ||\beta||}$

酉空间

设$V$是复数域上的线性空间，若对于$V$中任意两个元素$\alpha, \beta$，都定义了内积运算$(\alpha, \beta)$，且满足：

  1. $(\alpha, \beta) = \overline{(\beta, \alpha)}$
  2. $(k \alpha, \beta) = k(\alpha, \beta) = (\alpha, \bar k \beta)$
  3. $(\alpha + \beta, \gamma) = (\alpha, \gamma) + (\beta, \gamma)$
  4. $(\alpha, \alpha) \ge 0$，且$(\alpha, \alpha) = 0 \Leftrightarrow \alpha = \theta$

则称$V$是酉空间。

正定、半正定、负定

设$f = x^\mathrm HAx$是埃尔米特二次型。

  1. 若对于$\forall x \neq 0, x^\mathrm H A x > 0$，则称$f$是正定的。
  2. 若对于$\forall x \neq 0, x^\mathrm H A x < 0$，则称$f$是负定的。
  3. 若对于$\forall x \neq 0, x^\mathrm H A x \ge 0$，则称$f$是半正定的。
  4. 若对于$\forall x \neq 0, x^\mathrm H A x \le 0$，则称$f$是半负定的。
  5. 否则，称$f$是不定的。

商空间

$V$是数域$F$上的线性空间，$W$是$V$的子空间。对于$\alpha, \beta \in V$，若$\alpha - \beta \in W$，则称$\alpha$与$\beta$模$W$同余，记作$\alpha \equiv \beta \pmod W$。

模$W$同余是一种等价关系，因为它满足反身性、对称性、传递性。

定义模$W$的同余类$\alpha + W = \{\alpha + \beta | \beta \in W\}$，$\alpha$叫做这个同余类的一个代表。

同余类的性质：

  1. 同余类与选取的代表无关
  2. 两个模$W$的同余类或者完全相等，或者没有公共元素

按模$W$同余这一等价关系定义的等价类的集合$\overline V = \{\alpha + W | \alpha \in V\}$称为线性空间$V$的商集。

$\overline V$中元素定义加法与数乘运算：

  1. $(\alpha + W) + (\beta + W) = (\alpha + \beta) + W$
  2. $k(\alpha + W) = k\alpha + W$

$\overline V$中运算满足线性空间的七条公理，称为$V$的商空间，记作$V/W$。

$\dim V/W = \dim V - \dim W$

设$\sigma$是复数域$\mathbb C$上的线性空间$V$的一个线性变换，则$\sigma$的任意真不变子空间必包含在一个维数增加$1$的不变子空间中。

设$\lambda$是$\tau$的特征值，$\tau(\alpha + W) = \lambda(\alpha + W) = (\lambda\alpha + W)$。

于是$\sigma\alpha = \lambda \alpha + w, w \in W$。

令$U = L(\alpha \cup W)$。$\forall \gamma \in U, \gamma = k\alpha + \beta, \beta \in W$。

$\therefore \sigma \gamma = \sigma(k\alpha + \beta) = k(\lambda \alpha + w) + \sigma \beta = k\lambda \alpha + kw + \sigma \beta \in U$

子空间的直和
